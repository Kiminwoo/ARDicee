# ARDicee
Simple augmented reality app using SceneKit and ARKit

## Requirements
- Xcode13.2.1 +
- swift5.5 +
- iOS15.2 +

## Start
```
git clone https://github.com/donggyushin/ARDicee.git
cd ARDicee
xed ARDicee/
command + R
```

## Features
- Tap the screen to put a dice in ARWorld
- Shake the device to roll the dices
